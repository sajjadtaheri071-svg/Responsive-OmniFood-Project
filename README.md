# 🍔 Responsive OmniFood Project

A modern and fully responsive food landing page inspired by a food delivery service.  
Built with clean HTML and CSS, focusing on user experience, layout structure, and responsive design.

---

## ✨ Features

- 📱 Fully responsive (mobile, tablet, desktop)
- 🎨 Clean and modern UI design
- ⚡ Smooth layout and spacing
- 🧭 Easy navigation structure
- 💡 Built with best practices

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- Flexbox  
- CSS Grid  
- Media Queries  

---

## 📱 Responsive Design

This project is designed with a **mobile-first approach**, ensuring a seamless experience across:

- 📱 Mobile devices  
- 📲 Tablets  
- 💻 Desktop screens
